# Моя конфигурация 1С

Учебный проект на платформе 1С:Предприятие 8.3.

## 1) Элементы конфигурации

### Справочники:
- Формирование списка организаций.
- Размещение информации о товарах.
- Филиалы и менеджеры.

### Формы справочников. Перечисления:
- Создание перечисления именем «Категории».
- Редактирование справочника «Фирмы». Создание и редактирование форм справочника.
- Редактирование справочника «Города». Создание формы элементы справочника.
- Редактирование справочника «Менеджеры». Создание и редактирование формы справочника.
- Редактирование справочника «Товары». Создание формы списка справочника.
- Редактирование справочника «Товары». Создание формы элементы справочника.

### Документы. Формы документа:
- Учет поступлений товаров.
- Учет продажи товаров.
- Создание формы документа «Поступление товаров».
- Создание формы документа «Продажа товаров».

## 2) Обработка документов и регистры

### Программирование форм документов:
- Программирование расчетов при учете поступлений товаров.
- Программирование расчетов при учете продажи товаров.

### Регистры сведений:
- Создание регистра сведений «Цены поступления» для определения закупочной цены товара, в котором будут храниться цены, развернутые во времени.
- Редактирование документа «Поступление товаров» с учетом появления регистра сведений.
- Использование общего модуля для получения данных из регистра сведений.
- Программирование формы документа «Продажи товаров» с учетом регистра сведений.

### Регистры накопления. Движения по регистрам:
- Создание регистра накопления «КоличествоТоваров».
- Редактирование документа «ПоступлениеТоваров» с учетом появления регистра накопления.
- Редактирование документа «ПродажаТоваров» с учетом появления регистра накопления.
- Создание оборотного регистра накопления «ПродажиПоФилиалам» для учета реализации товаров по филиалам.

## 3) Механизмы сохранения и обработки данных

### Печатная форма документа. Ввод на основании. Журнал документов:
- Создание документа «ЗаказыКлиентов» для реализации механизма «Ввод на основании».
- Создание механизма «Ввод на основании».
- Создание печатной формы документа «Поступление товаров».
- Создание журнала документов.

### Реализация запросов в обработках:
- Создание обработки для получения данных о сотрудниках фирм.
- Создание обработки для получения данных по остаткам товаров.

### Процедура «Обработка проведения»:
- Редактирование обработки проведения документа Продажа товаров для реализации контроля отрицательных остатков.

## 4) Управляемые формы 1С 8.3

### Реквизиты управляемой формы 1С:
- Настройка формы справочника.
- Настройка формы списка справочника.
- Настройка формы документа «Приготовление продуктов».
- Настройка формы документа «Приготовление продуктов».
- Заполнение документа на основании выбранных порций «Приготовление продуктов».

### Обработчики управляемых форм 1С:
- Процедура ОбработкаИзмененияПродукта.
- Процедура ПродуктПриИзменении.
- Процедура ОчиститьТабЧастьПродукты.

## 5) Отчеты и обработки

### Создание отчетов и обработок:
- Проектирование печатного макета.
- Проектирование обработки.

### Создание отчетов с помощью механизма СКД:
- Проектирование отчета с помощью СКД, вид отчета «Список».
- Проектирование отчета с помощью СКД, вид отчета «Диаграмма».

### Внешние отчеты и обработки:
- Создание внешнего отчета.
- Создание внешней обработки.

## 6) Импорт данных

### Создание механизма импорта данных:
Задача заключается в импорте данных для справочников: «Товары», «Города» и «Фирмы»:
- Импорт фотографий товаров.

### Подсистемы. Стили. Настройка интерфейса:
- Создание подсистем.
- Создание корпоративного стиля.
- Конструирование рабочего пространства.
- Применение изображений в интерфейсе конфигурации.

## 7) Динамический список и Дерево значений

## Технологии
- Платформа 1С:Предприятие 8.3
- Управляемые формы
- Общие модули
- Регистры сведений и накопления
- СКД (Система компоновки данных)
